About

This is an online tool for creating your portfolio exportable to XML. Builder form is generated directly from XML Schema that is used for validation of output XML files. Filled out portfolio can be then exported as HTML5. You can supply your own XML Schema with few limitations from form generation and also provide custom XSLT Transofrnations to export validated form. You can check out project's source on github here and feel free to open any issues with requests or questions.
